# Лендинг «Фермерская усадьба · Поседкино»

Премиальный одностраничник для инвесторов/покупателей объекта в Мытищинском округе.
Один самодостаточный файл — `index.html` + папка `assets/`. Никакой сборки не нужно.

## Как посмотреть
Открыть `index.html` в браузере (двойной клик). Всё.

## ‼️ НЕ отправляй лендинг файлом (важно!)
Если переслать `index.html` в мессенджере, на айфоне он откроется во встроенном просмотрщике (Quick Look),
где **не работает JavaScript и не подтягиваются картинки из папки** — часть эффектов и форма отвалятся,
а если прислать файл без папки `assets/` — пропадут и фото/схема.
**Правильно — выложить на хостинг и отправлять ССЫЛКУ** (см. ниже). Тогда открывается полноценный сайт в Safari/Chrome.
(Сам файл теперь тоже деградирует аккуратно — текст виден даже без JS, — но полноценно он живёт только по ссылке.)

## Как выложить в интернет (бесплатно, 2 минуты)
1. Зайти на **netlify.com** (или **vercel.com**) → «Add new site» → «Deploy manually».
2. Перетащить папку `Lending` целиком.
3. Готово — получите ссылку вида `xxx.netlify.app`. Можно привязать свой домен.

## ⚠️ Что обязательно сделать перед показом инвесторам

### 1. Подключить форму → Google Sheets (рекомендуется)
Заявки будут падать строками в Google Таблицу, доступ к которой можно раздать кому нужно.
Пошаговая инструкция — в файле **`google-apps-script.gs`** (в нём же готовый код). Кратко:
1. Создать Google Таблицу → Расширения → Apps Script → вставить код из `google-apps-script.gs`.
2. Развернуть как «Веб-приложение», доступ «Все» → скопировать URL.
3. Вставить URL в `index.html` в строку `const SHEETS_ENDPOINT = "..."` → `git commit` + `git push`.

Пока URL не вставлен — форма в демо-режиме (показывает «спасибо», но никуда не шлёт).

Альтернативы, если нужно: **Telegram-бот** (мгновенный пинг в личку/группу — можно добавить сверху к Sheets),
**Formspree**, **Resend** (на почту). Доступно по запросу.

### 2. Вставить реальные фото
По всему сайту стоят элегантные **фото-слоты** с подписью «что снять» (совпадают с разделом 6 ТЗ — это готовый шот-лист).
Каждый слот `<div class="shot">…</div>` заменить на `<div class="shot"><img src="assets/имя.jpg" alt="…"></div>`.

**Приоритет съёмки:**
- Дрон сверху: весь массив, дом, поля, лес → в Hero.
- Длинный фасад дома, дом на закате, терраса с длинным столом, печь с огнём, балки/высокий потолок, вид из окон → блок «Дом».
- Клубничное поле, люди на самосборе, лукошки → блок «Ягода» / «Сценарии».

### 3. Заменить фон Hero на дрон-кадр
Сейчас фон Hero — благородный природный градиент (осознанная заглушка, без фейковых зданий).
Когда будет дрон-фото: в CSS у `.hero__bg` добавить
`background: url("assets/hero.jpg") center/cover;` — текстовый оверлей уже настроен.

### 4. Уточнить локацию
В блоке «Локация» расстояния поданы обтекаемо (точных км/времени в ТЗ не было). Впиши реальное время в пути.

## Что НЕ трогали намеренно (по ТЗ)
- **Деньги** поданы сдержанно: «финмодель по запросу», без крупных цифр прибыли на виду. Так безопаснее юридически.
- Все зоны развития помечены как **«потенциальный сценарий»**, а не существующая инфраструктура.
- Стоит `<meta name="robots" content="noindex">` — сайт закрыт от поиска (это закрытое предложение). Убрать, если решишь сделать публичным.

## Стек / дизайн
- Чистый HTML + CSS + чуть JS (без фреймворков, без зависимостей).
- Шрифты: Cormorant Garamond (заголовки) + Manrope (текст), Google Fonts.
- Палитра: молочный / древесный / графит + глубокий зелёный, акцент бронза, мягкий ягодный.
- Отзывчивость: десктоп / планшет / мобайл проверены. Анимации уважают `prefers-reduced-motion`.
